# MAHANEY CRM - Outreach Tool for Microsoft Excel

Multi-channel outreach management system (Email, Text/SMS, LinkedIn) built on a Microsoft Excel/OneDrive spreadsheet.

## Document Map

```
/docs
  executive-plan.md                  # Executive plan - vision, goals, timeline, milestones
  /product-specs
    email-component.md               # MSAL auth, Azure AD, Microsoft Graph API, email sync pipeline
    text-component.md                # chat.db/SQLite reader, SMS sync pipeline, watermark logic
    linkedin-component.md            # LinkedIn scraper, profile data mapping, connection status
  /page-specs
    tracker-page.md                  # Main tracker tab - status engine, priority ranking, column layout
    outreach-page.md                 # Outreach tab - send flow, status tracking, contact/company fields
    template-page.md                 # Templates tab - fill text templates for outreach messages
```
